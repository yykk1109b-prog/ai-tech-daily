## OpenAIがGPT-5.4をリリース — PC操作ネイティブ対応でエージェント性能強化

> 元記事: [GIGAZINE](https://gigazine.net/news/20260306-openai-gpt-5-4/)

OpenAIが2026年3月5日、プロフェッショナルな業務に特化して設計された最新フロンティアモデル「GPT-5.4」をリリースしました。このモデルはChatGPT、API、Codexを通じて展開され、推論やコーディング、自律的なエージェントとしてのワークフローにおける成果が一つのモデルに集約されています。同時に、より高度な推論を行う「GPT-5.4 Thinking」と複雑なタスクで最高の性能を発揮する「GPT-5.4 Pro」も発表されています。PC画面を認識して操作する「computer use」機能が統合されたのが特徴です。

### エージェント機能のネイティブ対応

GPT-5.4は、OpenAIにとって初めて「computer use」機能をネイティブに統合したモデルです。これまで別途必要だったツールや統合が不要になり、モデル単体でPC操作を実行できます。マウス操作やキーボード入力をAIが代行できるため、スプレッドシートの作成、ドキュメントの編集、メールの送信など、複数アプリケーションにまたがるタスクを自動化できます。

### プロフェッショナルワークへの最適化

「GPT-5.4 Pro」は知識労働や専門的な実務において高い能力を発揮するように構築されています。推論能力が向上し、複雑な文書分析やレポート作成の精度が高まっています。一方、「GPT-5.4 Thinking」は思考プロセスへの介入が可能で、AIがどう結論に至ったかを可視化できます。

## Anthropicが米国防総省から「サプライチェーンリスク」に指定 — 法廷闘争へ

> 元記事: [The Verge AI](https://www.theverge.com/ai-artificial-intelligence/890347/pentagon-anthropic-supply-chain-risk)

Claudeの開発で知られるAI企業のAnthropicがアメリカ国防総省によって「国家安全保障に対するサプライチェーンリスク」に指定されました。この決定は数週間にわたる交渉の決裂、公開の最後通牒、訴訟の脅しを経て正式化されました。Anthropicは法廷闘争を宣言しており、CEOのDario Amodei氏は米国防総省の指定を法廷で争う意向を表明しています。

### 指定の背景と影響

サプライチェーンリスク指定は、米国防総省の契約調達における重要なラベルです。この指定を受けると、国防総省との契約取得が事実上不可能になります。一方で、興味深いことに米国防総省は現在もイランでの運用においてAnthropicのAIを使用しています。

### 業界への波及

この決定はAI業界において前例のない展開です。Anthropicは米国企業でありながら、自国の国防総省からリスクと見なされる最初のケースとなりました。AI企業と政府の関係性において、契約条件や受け入れ可能なポリシーの境界線がどこにあるかを巡る議論が激化しています。

## Google Workspace CLI登場 — コマンドラインから主要サービスを一括管理

> 元記事: [GIGAZINE](https://gigazine.net/news/20260306-google-workspace-cli-gwc/)

Googleが2026年3月6日、Workspaceの主要サービスを網羅的に管理できる新たなコマンドラインツール「Google Workspace CLI(gws)」をリリースしました。このツールは、GoogleドライブやGmail、Googleカレンダー、Googleスプレッドシート、Googleドキュメント、Google Chat、さらに管理コンソールに至るまで、広範なアプリケーションを一括して1つのコマンドラインで操作することを可能にします。AIエージェントのスキルも搭載されています。

### 開発者が押さえておくべきポイント

`gws`コマンドはPythonで作成されたCLIツールで、pipでインストールできます。設定ファイル`.gws/config.yaml`で認証情報やデフォルト設定を管理します。コマンド例としては、`gws files list`でドライブ内のファイルを一覧表示、`gws mail send --to@example.com`でメール送信、`gws drive upload file.txt`でファイルアップロードなどが可能です。

### 自動化ワークフローへの統合

このCLIツールの最大の価値は、AIエージェントやシェルスクリプトとの連携です。GPT-5.4のようなAIエージェントがCLIを経由してGoogle Workspaceのサービスを操作できるため、複雑なワークフローをプログラム的に実現できます。例えば、メールを受信 → 情報を解析 → カレンダーに予定を追加 → ドライブに資料を保存、といった一連の処理を自動化できます。

## Netflixがベン・アフレックのAIスタートアップ買収 — 映画製作支援AI

> 元記事: [TechCrunch AI](https://techcrunch.com/2026/03/05/netflix-buys-ben-afflecks-ai-filmmaking-company-interpositive/)

バットマンを演じた俳優のベン・アフレック氏が2022年に創業したAIスタートアップ「InterPositive」をNetflixが買収しました。InterPositiveはAI俳優や合成演技を作るのではなく、制作チームが自分たちの作品の映像素材を扱う際に役立つモデルを作成しました。ポストプロダクションにおける編集作業を支援するのが目的です。InterPositiveの現在16名のエンジニアチーム全員がNetflixにそのまま参加し、アフレック氏もシニアアドバイザーとして関わり続けるとのことです。

### なぜNetflixは映画制作AIを買収したのか

ストリーミング戦争において、コンテンツ制作の効率化は競争力を左右する重要な要素です。InterPositiveの技術は、膨大な映像素材を高速に処理し、編集作業を支援できます。これによりNetflixは制作コストを削減しつつ、より短いサイクルで新しいコンテンツを提供できるようになります。ハリウッドにおいてAI導入が進む中、制作現場での実用的なツールを持つことが差別化要因になります。

### 業界の反応

ハリウッドはAIに対して懐疑的な姿勢も強くありますが、今回の買収は「AIが人間を置き換える」のではなく「人間の作業を効率化する」方向性を示しています。俳優が創業したスタートアップを大手が買収する事例は珍しく、エンターテインメント産業におけるAIの役割が変化しつつある象徴的な出来事と言えます。

## ローカルで動作する動画生成AI「LTX-2.3」登場 — 無料PCアプリも公開

> 元記事: [GIGAZINE](https://gigazine.net/news/20260306-ltx-2-3-video-generation-ai/)

動画生成AIモデル「LTX-2.3」が2026年3月6日に登場しました。LTX-2.3はローカルで実行可能なオープンモデルとして公開されており、前世代モデルのLTX-2と比べて「プロンプト理解力向上」「画像から動画を生成する際の一貫性向上」「音声品質向上」といった機能強化が盛り込まれています。加えて、WindowsとmacOS向けのデスクトップアプリ「LTX Desktop」も無料で公開されています。

### 技術的なポイント

LTX-2.3はLightricks社が開発した動画生成モデルで、Hugging Faceで公開されています。ローカル実行が可能なため、GPUを持つPCであれば外部サーバーに依存せずに動画を生成できます。モデルのサイズは約3GBと比較的コンパクトで、消費者向けGPUでも推論が可能です。

### 今日から試せるコマンド例

```bash
# Hugging Faceからモデルをダウンロード
pip install ltx-video

# テキストから動画を生成
python -m ltx.video.generate --prompt "夕焼けのビーチを歩く猫" --output video.mp4

# LTX Desktopアプリを起動（Windows/macOS）
# GUIでテキストや画像を入力して動画を生成可能
```

### オープンモデルとしての意義

LTX-2.3がオープンモデルとして公開されたことは、開発者やクリエイターにとって大きなメリットです。APIコストを気にする必要がなく、生成した動画を商用利用しやすいライセンスが提供されています。また、モデルをローカルで微調整できるため、特定のスタイルやニーズに合わせたカスタマイズも可能です。
