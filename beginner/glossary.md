# AI用語集 - 押さえておきたい10語

## 基本用語

### 生成AI
文章や画像を作り出すAIのこと。
例: ChatGPT（文章）、Midjourney（画像）

### LLM（大規模言語モデル）
大量のテキストデータを学習したAIモデル。
人間の言葉を理解し、自然な文章を生成できます。

### プロンプト
AIへの指示や質問のこと。
良いプロンプトを書くと、より良い結果が得られます。

### チャットボット
テキストで会話できるAIプログラム。
カスタマーサポート、情報提供などに使われます。

## 仕組みに関わる用語

### 機械学習
AIがデータから自動的にパターンを学ぶ仕組み。
多くのAIはこの技術で作られています。

### ニューラルネットワーク
人間の脳の構造を模したAIの仕組み。
複雑なパターン認識ができます。

### 学習データ
AIが学習するためのデータ。
質と量がAIの性能に大きく影響します。

## 使い方に関わる用語

### API
アプリケーションからAI機能を使うための窓口。
開発者が自作アプリにAIを組み込めます。

### ファインチューニング
既存のAIモデルを特定の用途に合わせて調整すること。
企業の専用AIなどで使われます。

### RAG（検索拡張生成）
最新の情報を検索して、AIの回答に活用する仕組み。
AIの知識不足を補えます。
