# AI用語集 - 押さえておきたい基本用語

## 基本用語

### 生成AI
文章や画像、音声などを新しく作り出すAIのこと。
- 例: ChatGPT（文章生成）、Midjourney（画像生成）

### LLM（大規模言語モデル）
大量のテキストデータを学習したAIモデル。「Large Language Model」の略。
- ChatGPT、Claudeなどが使われている技術

### プロンプト
AIへの指示や質問のこと。
- 例: 「今日の天気を教えて」「この文章を要約して」

### チャットボット
AIと対話形式でやり取りができるツール。
- 文字で会話して質問・回答ができる

## その他よく使う用語

### GPT
OpenAI社が開発したAIモデルのシリーズ名。「Generative Pre-trained Transformer」の略。

### ゼロショット
プロンプトだけで、事前の例や説明なしでAIにタスクをさせること。

### チューニング
特定の用途に合わせてAIモデルを微調整すること。

### RAG（ラグ）
「Retrieval-Augmented Generation」の略。外部データを参照して回答精度を高める技術。

### トークン
AIが扱うテキストの単位。単語や文字の一部に相当する。

### コンテキスト
AIが現在の会話で記憶している情報のこと。

### API
外部のアプリやサービスと連携するための仕組み。

### ファインチューニング
特定のデータで追加学習させること。

### コンテキストウィンドウ
AIが一度に記憶できる情報量の上限。

## これだけは覚えておきたい

最初は**「生成AI」「プロンプト」「LLM」**の3つだけ覚えておけば十分です。
他の用語は必要になったときに調べれば大丈夫です。
