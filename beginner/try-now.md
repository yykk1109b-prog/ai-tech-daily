# さっそく試してみる

理論は後で良いです。まずはAIを使ってみましょう！

## 今すぐ試せるツール（アカウント不要）

### 1. Claude Web
- URL: https://claude.ai
- 使い方: テキストボックスに質問を入力するだけ
- 例: 「東京の観光スポットを5つ教えて」

### 2. ChatGPT
- URL: https://chat.openai.com
- 特徴: 文章生成、翻訳、要約など幅広い用途に対応

## 30秒でできる例

1. ツールを開く
2. 「今日の晩御飯のおすすめを3つ教えて」と入力
3. Enterキーを押す

これだけでAIが提案してくれます。

## プロンプトのコツ

AIへの指示（プロンプト）は具体的であるほど良い結果が得られます。

- 悪い例: 「メールを書いて」
- 良い例: 「上司に休暇申請をする丁寧なメールを書いてください。期間は4月10日〜14日です」
