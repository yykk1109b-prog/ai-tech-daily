# さっそく試してみる

理論は後で良いです。まずはAIを使ってみましょう！

## 今すぐ試せるツール（アカウント不要）

### 1. Claude Web
- **URL**: https://claude.ai
- **使い方**: テキストボックスに質問を入力するだけ
- **例**: 「東京の観光スポットを5つ教えて」

### 2. ChatGPT
- **URL**: https://chat.openai.com
- **使い方**: アカウント登録後、会話画面でチャット

## 30秒でできる例

1. ツールを開く
2. 「今日の晩御飯のおすすめを3つ教えて」と入力
3. Enterキーを押す

たったこれだけで、AIが提案をしてくれます。

## よく使うプロンプト（質問の例）

| 目的 | プロンプト例 |
|------|-------------|
| メール作成 | 「上司に休暇を申請する丁寧なメールを書いて」 |
| 要約 | 「この文章を3行で要約して：（文章を貼り付け）」 |
| 翻訳 | 「この日本語を英語に翻訳して：（文章を貼り付け）」 |
| アイデア | 「新しいビジネスのアイデアを5つ出して」 |
| 学習 | 「Pythonを学ぶための学習計画を作って」 |
