# AIとは？ - 5分でわかる入門ガイド

## AIってそもそも何？

AI（人工知能）は、「コンピュータが人間のように考えるプログラム」のことです。

「人間のように考える」といっても、難しい数学やプログラミング知識は必要ありません。私たちが日常生活で使っているスマホやアプリの中に、既にAIは溶け込んでいます。

## 身近なAI例

あなたは今までに、こんな体験をしたことはありませんか？

- スマホで顔認証でロック解除した
- YouTubeで動画を観ていると、気に入った動画が次々とおすすめされた
- 文字入力で「予測変換」が出てきて、タイピングが楽になった
- 写真アプリで自動で人物が分類された

これらはすべて、AIの働きによるものです。

## AIができること

現在のAI（特に生成AI）は、以下のようなことができます：

| できること | 具体例 |
|-----------|--------|
| 文章を作成 | メール、記事、SNS投稿、レポートなど |
| 画像を生成 | 写真風の画像、イラスト、ロゴデザインなど |
| 要約・翻訳 | 長い文章を短くまとめる、多言語間翻訳 |
| アイデア出し | ブレインストーミング、コンテンツ企画 |
| Q&A | 質問に答える、情報検索 |
