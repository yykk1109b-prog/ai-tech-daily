# お問い合わせ

AI Tech Daily（以下「本サイト」）に関するご質問、ご要望、ご意見などは、以下の方法でお問い合わせください。

---

## GitHub Issues

本サイトは GitHub でオープンソース開発されています。技術的なバグ報告、機能の要望、コンテンツに関するご意見などは、GitHub Issues をご利用ください。

1. [ai-tech-daily リポジトリ](https://github.com/yykk1109b-prog/ai-tech-daily) にアクセス
2. 「Issues」タブをクリック
3. 「New Issue」をクリックして新しいIssueを作成

---

## お問い合わせの種類

| お問い合わせ内容 | 対応方法 |
|-----------------|---------|
| サイトの不具合報告 | GitHub Issues |
| 機能の要望・提案 | GitHub Issues |
| 記事内容に関するご意見 | GitHub Issues |
| 広告・提携のご相談 | GitHub Issues |
| その他全般 | GitHub Issues |

---

## 対応時間

通常、いただいたお問い合わせには 2〜3 営業日以内に返信いたします。ただし、お問い合わせの内容によっては、それ以上のお時間をいただく場合がございます。

---

## 注意事項

- 個人情報を含むお問い合わせは避けてください
- スパム、宣伝、攻撃的な内容の投稿は削除させていただきます
- 法的措置を伴うお問い合わせについては、直接弁護士にご相談ください
