# AI活用例

AIをどう活用しているか、実際の例を紹介します。

## 文章作成・メール対応

### メールの件名を考える

**ツール:** ChatGPT, Claude

**プロンプト例:**
```
プロジェクトの進捗状況報告メールの件名を、以下の要件で5案作ってください。

要件:
- チーム名: [チーム名]
- 期日: [日付]
- 重要度: [高・中・低]
```

### 要約

長い議事録を要約するのに役立ちます。

**プロンプト例:**
```
以下の議事録を100文字以内で要約してください。

[議事録を貼り付け]
```
