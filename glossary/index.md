# AI用語集

AIに関わる用語をまとめました。

## 基礎用語

### LLM（Large Language Model）
大量のテキストデータで学習した大規模言語モデル。

**代表例:**
- GPT-4, Claude Sonnet, Gemini

### 生成AI
文章、画像、音楽などを生成するAI。

**代表例:**
- ChatGPT（文章）
- Midjourney（画像）
- Suno（音楽）

### プロンプト（Prompt）
AIへの指示や質問のこと。

**例:**
```
明日の天気を教えてください
```

**良いプロンプトのポイント:**
- 具体的にする
- 目的を明確にする
- 文脈を提供する

### コンテキスト（Context）
AIとの対話履歴や参照情報のこと。

---

## 技術用語

### RAG（Retrieval-Augmented Generation）
検索拡張生成。最新の情報を検索してAIの回答に活用する技術。

### Fine-tuning
既存のAIモデルを特定の用途に合わせて調整すること。

### API（Application Programming Interface）
アプリケーションからAI機能を使うためのインターフェース。
