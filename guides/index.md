# AIの使い方

主要なAIツールの使い方をまとめました。

## ChatGPT

**公式サイト:** https://chat.openai.com

### 使い方

1. 公式サイトにアクセス
2. 「Sign up」でアカウント作成（無料）
3. ログインしてチャット画面を開く
4. 質問を入力して送信

### 初心者におすすめのプロンプト

- 「今日の天気を教えて」
- 「メールの件名を考えて3つ」
- 「この文章を要約してください」

## Claude

**公式サイト:** https://claude.ai

### 使い方

1. 公式サイトにアクセス
2. 「Try Claude」でチャット開始（無料、アカウント不要）
3. 質問を入力

### 特徴

- 安全で信頼性の高い設計
- 長い文脈での対話が得意
- プログラミングサポートが充実
